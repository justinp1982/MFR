# MFR


MFR Rapid Doc Tool - Quick Start Guide

What is this?
This is a high-speed documentation tool designed for Manual Therapy sessions. It runs entirely in your browser and saves data to your device automatically.

How to use it:

The Map: Tap body parts to select them.

Green Mode: Marks areas you Treated.

Red Mode: Marks areas of Pain/Restriction.

Note: You can switch modes using the buttons at the top.

The Details:

As you select muscles, the Middle Panel updates.

Tap the Technique (e.g., MFR) and Response (e.g., Improved ROM) buttons. These apply to the current session notes.

The Output:

SOAP Note: The text box automatically writes your "Subjective, Objective, Assessment, Plan" note. Click "Copy Text" to paste it into your EMR.
